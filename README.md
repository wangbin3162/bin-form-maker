# bin-form-maker

表单设计器,基于 bin-ui-design 制作的表单设计器模块，可提供快速界面搭建并进行动态渲染，支持自定义组件插入。

支持表单各种属性验证，动态验证，脚本验证，全局可定义脚本扩展操作和界面属性。灵活易用。
